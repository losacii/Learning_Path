### Record Terminal with ttyrec
    sudo apt install ttyrec

    ttyrec <rec.Title>
    <C-d> to stop

    ttyplay -s 4.0 <rec.Title>

### Hot keys
    <Tab>    | it's for Order complete
    C-l      | Clear Previous output
    C-u      | Delete input
    C-c      | Cancel operation
    C-a  C-e | move cursor to begin/end
